## 系统概述

OpenCode 桌面端应用是一个基于 Tauri 框架构建的跨平台桌面应用，支持 Windows、macOS 和 Linux。该应用采用了现代的混合架构，结合了：
- **前端**: Solid.js 构建的 Web 界面
- **后端**: Rust 编写的原生代码
- **Sidecar**: 集成的命令行工具作为本地服务器

### 核心特性
- 跨平台支持（Windows/macOS/Linux）
- 内置本地服务器（Sidecar）
- 自动更新机制
- 原生菜单系统（macOS）
- 深度链接支持
- WSL 集成（Windows）
- 显示后端选择（Linux）
- Markdown 解析
- 剪贴板图像读取

#### 初始化流程

```rust
async fn initialize(app: AppHandle) {
    // 1. 设置初始化通道
    let (init_tx, init_rx) = watch::channel(InitStep::ServerWaiting);
    
    // 2. 设置应用
    setup_app(&app, init_rx);
    
    // 3. 启动 CLI 同步任务
    spawn_cli_sync_task(app.clone());
    
    // 4. 生成 Sidecar 端口和密码
    let port = get_sidecar_port();
    let password = uuid::Uuid::new_v4().to_string();
    
    // 5. 启动 Sidecar
    let (child, health_check) = server::spawn_local_server(
        app.clone(), 
        hostname, 
        port, 
        password.clone()
    );
    
    // 6. 立即可用的 Sidecar 凭据（健康检查前）
    let (ready_tx, ready_rx) = oneshot::channel();
    let _ = ready_tx.send(ServerReadyData {
        url: format!("http://{hostname}:{port}"),
        username: Some("opencode".to_string()),
        password: Some(password),
    });
    
    // 7. 管理状态
    app.manage(SidecarReady(ready_rx.shared()));
    app.manage(ServerState {
        child: Arc::new(Mutex::new(Some(child))),
    });
    
    // 8. SQLite 迁移处理
    let needs_migration = !sqlite_file_exists();
    let sqlite_done = needs_migration.then(|| { /* ... */ });
    
    // 9. 加载任务（等待 SQLite 和健康检查）
    let loading_task = tokio::spawn({ /* ... */ });
    
    // 10. 显示加载窗口（如需要）
    let loading_window = if needs_migration && /* ... */ {
        Some(LoadingWindow::create(&app)?)
    } else {
        None
    };
    
    // 11. 立即创建主窗口
    MainWindow::create(&app)?;
    
    // 12. 等待加载完成
    let _ = loading_task.await;
    let _ = init_tx.send(InitStep::Done);
    
    // 13. 关闭加载窗口
    if let Some(loading_window) = loading_window {
        loading_window.close();
    }
}
```

## Sidecar 集成

### 架构

```
桌面应用
    ↓ Tauri 命令
Rust 后端
    ↓ 进程管理
Sidecar (opencode-cli serve)
    ↓ HTTP
本地服务器 (http://127.0.0.1:<port>)
    ↓ 前端 API 调用
Solid.js 前端
```

### 连接流程

1. **启动时**
   - Rust 后端找到空闲端口
   - 生成随机 UUID 作为密码
   - 启动 Sidecar 进程
   - 通过 oneshot channel 立即将凭据发送到前端

2. **健康检查**
   - 异步轮询 `/global/health`
   - 30 秒超时
   - 使用基本认证

3. **前端连接**
   - 接收 `ServerReadyData`（URL、用户名、密码）
   - 构建 `ServerConnection.Sidecar`
   - 传递给 `AppInterface`

4. **认证**
   - 所有请求使用基本认证
   - 用户名: `opencode`
   - 密码: 生成的 UUID

### 生命周期管理

```
应用启动
    ↓
生成端口 + 密码
    ↓
启动 Sidecar 进程
    ↓
立即向前端发送凭据
    ↓
前端开始使用（同时后台进行健康检查）
    ↓
健康检查成功
    ↓
正常运行
    ↓
应用关闭
    ↓
发送 kill_sidecar 命令
    ↓
终止 Sidecar 进程
    ↓
退出
```

## 总结

OpenCode 桌面端应用是一个设计精良的跨平台应用，具有以下优势：

1. **现代架构**
   - Tauri 框架提供安全、轻量的桌面体验
   - Solid.js 提供高性能前端
   - Rust 后端确保安全性和性能

2. **完整的功能集**
   - 本地 Sidecar 集成
   - 自动更新
   - 原生菜单
   - 深度链接
   - 多平台适配

3. **优秀的开发体验**
   - 类型安全的 IPC 绑定（tauri-specta）
   - 热重载开发
   - 统一的代码库

4. **生产就绪**
   - 完整的错误处理
   - 日志系统
   - 安全的进程管理
   - 专业的打包和签名

该桌面应用为用户提供了无缝的本地 AI 编码助手体验，同时保持了 Web 版本的所有功能。
